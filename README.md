# 🔥 핫딜사이트 - Next.js 버전

7개 주요 쇼핑몰의 실시간 핫딜 정보를 한 곳에서 확인할 수 있는 Next.js 기반 웹 애플리케이션입니다.

## 📋 프로젝트 개요

- **기술 스택**: Next.js 14, TypeScript, Tailwind CSS
- **아키텍처**: App Router 기반
- **반응형 디자인**: 모바일 퍼스트
- **접근성**: WCAG 가이드라인 준수

## 🚀 시작하기

### 1. 개발 서버 실행
```bash
npm run dev
```

### 2. 브라우저에서 확인
http://localhost:3000 (또는 3001) 접속

### 3. 프로덕션 빌드
```bash
npm run build
npm run start
```

## 🎯 주요 기능

### ✅ 완성된 기능
- **메인 페이지**: 30개 샘플 딜 데이터 표시
- **반응형 디자인**: 모바일/태블릿/데스크톱 대응
- **실시간 검색**: 상품명, 플랫폼명으로 검색
- **카테고리 필터**: HOT, NEW, 최저가, 마감임박 등 10개 카테고리
- **무한 스크롤**: 8개씩 페이지네이션
- **TypeScript**: 완전한 타입 안정성
- **접근성**: 스크린 리더, 키보드 네비게이션 지원

### 🔄 상호작용
- **검색**: 실시간 검색 결과 표시
- **필터링**: 카테고리별 딜 필터링
- **딜 클릭**: 상세 정보 알림 (실제 서비스에서는 상세 페이지로 이동)
- **더 보기**: 추가 딜 로딩
- **하단 네비게이션**: 홈, 검색, 찜, 내정보 탭

## 🎨 디자인 특징

- **모바일 퍼스트**: 최대 448px 폭에 최적화
- **그라데이션**: 각 딜 카드마다 고유한 배경 그라데이션
- **애니메이션**: 호버 효과, 페이드 인, 스케일 애니메이션
- **태그 시스템**: HOT, NEW, 최저가, 마감임박 등 시각적 태그
- **플랫폼 색상**: 각 쇼핑몰별 고유 브랜드 색상
