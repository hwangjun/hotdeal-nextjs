# 🔍 QA 체크리스트

## 📊 데이터 품질 체크

### ✅ 실제 데이터 사용
- [ ] 가격 정보가 RSS 제목과 일치하는가?
- [ ] 랜덤 생성 데이터가 완전히 제거되었는가?
- [ ] 평점/리뷰 수가 null 또는 "정보 없음"으로 표시되는가?
- [ ] 실제 이미지가 있는 딜은 이미지가 표시되는가?
- [ ] 이미지가 없는 딜은 "이미지 없음" 표시가 되는가?

### ✅ 가격 추출 정확성
- [ ] 뽐뿌: `(22,400원/무료)` → 22,400원으로 정확히 추출?
- [ ] 뽐뿌: `(2,780원/3,000원)` → 할인 정보 정확히 계산?
- [ ] 퀘이사존: 가격 정보 없는 경우 "가격 문의" 표시?
- [ ] 배송비 정보 (무료/유료/확인) 정확히 표시?
